# Wellcome Collection pattern library

## Clone this repo
- `git clone https://{THIS_REPO_NAME}.html`

## Install dependencies
- `cd {THIS_REPO_NAME}
- `npm install`

## Build the assets
- `npm run watch:js` and (in a separate terminal) `npm run watch:scss`

## Start a server
- from the root of this repo, run a server (`python -m SimpleHTTPServer`)
